There's some Groovy code below that illustrates how to do this.

After the first call of useFilter(), the filter model is fixed and can be applied to new datasets.

Cheers,
Eibe

import weka.core.converters.ConverterUtils.DataSource
import weka.core.Instances
import weka.filters.supervised.attribute.AddClassification
import weka.filters.Filter
import weka.classifiers.bayes.NaiveBayes

data = DataSource.read("/Users/eibe/datasets/UCI/iris.arff")
data.setClassIndex(data.numAttributes() - 1)
 
// Set up the filter
addClass = new AddClassification();
addClass.setClassifier(new NaiveBayes());
addClass.setRemoveOldClass(true);
addClass.setOutputClassification(true);
addClass.setInputFormat(data);

// Build the filter model on the training data, discarding the output
Filter.useFilter(data, addClass);

// Create some "test data"
for (inst in data) {
 inst.setClassMissing();
}

// Filter the test data
data = Filter.useFilter(data, addClass);

println data.toString()
